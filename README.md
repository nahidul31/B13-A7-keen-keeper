# 🌿 KeenKeeper — Keep Your Friendships Alive

KeenKeeper is a simple and modern friendship management app built with **Next.js**.  
It helps you track friends, manage interactions (Call, Text, Video), and view all activities in a clean timeline and analytics dashboard.

---

## 🚀 Live Demo

## 🔗 https://b13-a7-keen-keeper-ten.vercel.app/

## 📌 About the Project

KeenKeeper helps you stay connected with your friends by:

- Managing friend list from JSON data
- Tracking interactions like Call, Text, and Video
- Showing friend details in a clean UI
- Displaying all activities in a timeline
- Visualizing data using charts

---

## 🛠️ Tech Stack

- Next.js
- Tailwind CSS
- DaisyUI
- Recharts
- Lucide React Icons
- React Toastify
- Vercel

---

## ✨ Features

### 👥 Friends System

- Friend data from JSON file
- Shows name, photo, tags, status, bio, email
- Clickable cards for details page

---

### ⚡ Timeline System

- Tracks Call, Text, Video interactions
- Auto updates on user actions
- Filter by interaction type

---

### 📊 Analytics

- Pie chart for interaction overview
- Simple and clean data visualization
